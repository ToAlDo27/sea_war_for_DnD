# Ship Status для Owlbear Rodeo

Расширение для Owlbear Rodeo. Показывает и редактирует ХП, КД, скорость, МЭ и ресурсы корабля.

Если расширение открыто внутри комнаты Owlbear, данные синхронизируются через metadata комнаты. Все игроки, у которых открыта панель, видят одинаковый статус и могут менять его.

## Онлайн-версия

Добавьте расширение в Owlbear по ссылке:

```text
https://raw.githack.com/ToAlDo27/sea_war_for_DnD/main/owlbear-status-extension/manifest.json
```

Этот адрес берёт файлы прямо из ветки `main` на GitHub и отдаёт их по HTTPS с корректным типом `text/html` для панели расширения. Внутри manifest используются абсолютные HTTPS-ссылки на панель и иконку, чтобы Owlbear не ошибался при открытии вложенной папки.

## Запуск

Локальный запуск нужен только для разработки и проверки до публикации.

1. Запустите `start-extension.bat`.
2. В Owlbear Rodeo откройте профиль и добавьте расширение по ссылке:

```text
http://localhost:5173/manifest.json
```

Или:

```text
http://127.0.0.1:5173/manifest.json
```

3. Включите расширение в комнате.

## Импорт из конструктора

В основном конструкторе экспортируйте JSON, затем вставьте его в поле импорта внутри расширения.

Расширение забирает:

- имя корабля;
- день и игроков;
- текущие ХП и базовые параметры;
- ресурсы;
- текущую МЭ;
- генерацию и емкость МЭ из установленных генераторов/батарей.

Дальше данные хранятся в localStorage самого расширения.
