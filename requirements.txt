# =====================================================================
#  ТРЕБОВАНИЯ / REQUIREMENTS
#  Верфь Нового Мира — конструктор корабля для D&D

#  ВНИМАНИЕ: это проект на Node.js (React + Vite), а НЕ на Python.
#  Зависимости НЕ ставятся через pip. Этот файл — справочный список
#  требований. Реальная установка выполняется одной командой:

# ---------------------------------------------------------------------
#  СИСТЕМНЫЕ ТРЕБОВАНИЯ
# ---------------------------------------------------------------------
#  Node.js  >= 18.0   (рекомендуется 20+; проверено на 24.16.0)
#  npm      >= 9       (проверено на 11.13.0)
#  ОС:      Windows / macOS / Linux
#  Браузер: любой современный (Chrome / Edge / Firefox / Safari)
#           с поддержкой ES2020, localStorage и HTML5 drag-and-drop.
#  Сервер НЕ требуется — приложение полностью клиентское.
#
# ---------------------------------------------------------------------
#  УСТАНОВКА И ЗАПУСК
# ---------------------------------------------------------------------
#      npm install        # установить зависимости (создаёт node_modules/)
#      npm run dev         # дев-сервер: http://localhost:5173
#      npm run build       # production-сборка в dist/
#      npm run preview     # предпросмотр собранной версии
#      npm run typecheck   # проверка типов TypeScript
